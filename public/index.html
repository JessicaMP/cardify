¿Qué es Cardify?


Cardify es un plugin que transforma las imágenes semántica y dinámicamente; permite que nuestras imagenes sean correctamente colocadas dentro de una etiqueta figure junto con una etiqueta figcaption que contiene una descripción que es la misma que se muestra en el alt de la imagen.Cardify tambien crea para las imagenes un efecto de hover que muestra el alt de la imagen correspondiente.


Uso


Para tener acceso al plugin Cardify solo necesitas descargar el api cardify y enlazarlo en tu documento index.html.Tambien necesitarás enlazar la libreria de JQuery previamente"





Déspues debes colocar la clase "container-img" a cualquier elemento html que contenga elementos img con su atributo alt respectivamente




Listo! tus imagenes perfectamente semánticas y dinámicas en menos de 1 minuto
